# Movies
A demo Android app with: 
 * Kotlin Flow. 
 * Coroutines. 
 * Jetpack Components: Paging, Navigation, WorkManager, Hilt. 
 * Android KTK.
 * Clean Architecture.
 

Description
------------  
Find movies by name. See the details of the movie chose. You can add it as a favorite. List all the favorite movies you added.
